Circle Bambu Monitor

Install the ESP32-C3 firmware directly from your browser.

  1. Use Chrome or Edge on desktop.
  2. Connect the ESP32-C3 with a USB data cable.
  3. Close Arduino IDE or any serial monitor using the port.
  4. Click install and select the ESP32-C3 serial port.

This browser does not support Web Serial. Use Chrome or Edge on desktop.

Web flashing requires HTTPS or localhost.

After flashing, connect to WiFi CBM with password circlebambu, then open 192.168.4.1.
